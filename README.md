# ThreadVerse 🗣️

A text-only, community-driven discussion platform built with Node.js, Express, and PostgreSQL. Inspired by Reddit, ThreadVerse enables users to create communities, share posts, engage in threaded discussions, and vote on content.

## Overview

ThreadVerse is a lightweight discussion platform that emphasizes community-driven conversations. Users can:
- Create and manage communities (similar to subreddits)
- Post text-based discussions to communities
- Comment and reply to posts with nested threading
- Vote on posts and comments (upvote/downvote)
- Join and participate in communities of interest

**Key Features:**
- ✅ Text-only content (no media)
- ✅ Community-based organization
- ✅ Nested comment threads
- ✅ Voting system
- ✅ User authentication
- ✅ JWT-based session management
